Extendables - An Alternative to Mix-ins¶

Extendables are a way to extend the functionality of an object. So, for example, if you’ve created a book object and you want it to be read able, you’d instantiate the Readable extendable and set it to the readable property on the book object.

class avalon.Readable(prefix='', content=None)¶

Extendable to make object readable, i.e., you can use read on it.

Parameters:

  • prefix – To be used internally.

  • content – The text to be displayed when read is used.

class avalon.Surface¶

Extendable to make put [direct object] on [indirect object] work. Use this on the indirect object.
