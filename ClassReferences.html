Text Adventures¶

class avalon.TextAdventure(game_name: str, author_name: str, game_description: Optional[str] = None)¶

Class to create a text adventure. Ideally, you would create a subclass of this, and pass that to avalon.run.

avalon.TextAdventure.__init__(self, game_name: str, author_name: str, game_description: Optional[str] = None)¶

Create a new text adventure. Ideally, you would create a subclass of this, and pass that to avalon.run.

Parameters:

  • game_name – The name of the text adventure. (E.g. “The Bookshelf”)

  • author_name – The author of the game, probably you. (E.g. “Andrew Plotkin”)

  • game_description – Optional. The description of the game.
